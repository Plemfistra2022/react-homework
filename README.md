Rendering Lists (map)
Цель: Отрисовать список однотипных данных (массив объектов) в виде компонентов.

Ключи (key): При использовании map обязательно передаем key={index} (или ID), чтобы React понимал, какой элемент обновился.

📁 Архитектура проекта
src/models/ — типизация данных (интерфейсы).
src/data/ — файлы с данными (массивы объектов).
src/img/ — локальные изображения.
src/components/ — визуальные компоненты для отображения данных.

Что важно помнить:
Путь к картинке в объекте данных должен быть правильным относительно корня (например, src/img/milk.jpg).

В компоненте MyProduct мы принимаем объект product целиком через пропсы и типизируем его созданным интерфейсом IProduct.
